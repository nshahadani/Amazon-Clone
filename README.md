## Amazon UI Clone
A high-fidelity visual recreation of the Amazon homepage and product interface. This project focuses on pixel-perfect layout design and clean, semantic CSS architecture.

### Features
Pixel-Perfect UI: Replicated the iconic Amazon header, navigation bar, and multi-column layouts.

Hover Effects: Interactive navigation menus and product cards with smooth transitions.

Custom Icons & Assets: Integration of FontAwesome and optimized images for a realistic look.

Clean Codebase: Written with semantic HTML5 for better accessibility and SEO.

### Built With
HTML: Semantic structure for better accessibility.

CSS: Advanced styling including Flexbox, Grid, and Custom Properties (Variables).
